##委托模式
每个子元素都绑定相同事件时，可以把事件委托给父元素;还可以处理未来新生成的元素
```
ul.onclick = function(e){
    var e = e || window.event,
        tar = e.target || e.srcElement;
        if(tar.nodeName.toLowerCase() === 'li'){
            tar.style.backgroundColor = 'gray';
        }
}
```
